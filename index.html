洋葱煎蛋

Posted on 2021-12-05 Edited on 2021-12-13 In 热菜

备料

无特殊食材

做法

  1. 洋葱切丁

  2. 按照一个洋葱配4个鸡蛋的比例，准备鸡蛋

  3. 锅内烧热，少量油润锅，下入洋葱炒熟，然后倒入蛋液，煎成饼翻面煎熟即可。

备注

本菜简单，营养丰富，打散的鸡蛋外酥里嫩。煎成饼状，可以撒孜然，可以刷酱，可以做蛋汤，可以吊高汤。可以配面条，可以卷煎饼/大饼。适配性极强。

tips

锅底撒少量盐，可以进一步减少粘锅的概率。

芹菜鸡丁

Posted on 2021-11-29 In 热菜

备料

芹菜、鸡脯肉、孜然粉、豆干（可选）

做法

鸡脯肉切丁，用盐、白胡椒、一点点料酒腌制；

芹菜洗净切丁备用；五香豆干（optional）切丁；

大量葱姜蒜切丁或片；

国内烧油，炸五香豆干，捞出控油；

下入油烧热，下入鸡丁滑熟；

控油，留下鸡丁，加入花椒、葱姜蒜炒香；

稍微加入一点水，烹入料酒、生抽，稍微烹制；

加入切好的豆干、芹菜、炒熟；

胡椒粉、孜然粉调味，根据咸淡加入盐，根据口味加入辣椒粉，关火，出锅。

备注

无

葱油拌面

Posted on 2021-11-14 In 主食

备料

无特殊食材

做法

  1. 葱叶切丝，冷油下锅，小火炸干；

  2. 生抽：老抽 = 1：1，加入糖，酌情放入少量蚝油，加少许水，搅匀制成味汁；

  3. 葱油转最小火，倒入味汁，小火煮开至冒大泡即可关火；

  4. 煮面，捞出沥干，浇入葱油拌匀即可食用。

备注

无

tips

无

口水鸡

Posted on 2021-11-09 In 冷菜

备料

主料：鸡腿肉

调料：红油

做法

  1. 鸡腿冷水下锅，加入葱姜料酒、八角、花椒煮熟；
  2. 蒜泥、葱花、盐、一点点生抽、一点点醋、香油、花椒面，加入红油；
  3. 倒入煮鸡腿的汤，拌匀；
  4. 鸡腿切好装碗内，倒入拌匀的调味料即可。

备注

倒入鸡汤只要一点点，能沾满食材即可；主要还是吃红油

tips

可以加入熟花生碎或熟芝麻，丰富口感。

酸辣白菜

Posted on 2021-11-01 In 热菜

备料

大白菜

无特殊调料

做法

  1. 葱姜蒜，干辣椒爆香锅底
  2. 下白菜稍微翻炒
  3. 先加酱油、盐、水烧熟
  4. 加醋，再勾琉璃芡
  5. 撒一点香葱段出锅

备注

加淀粉是勾芡是琉璃芡，不要太浓稠了。

tips

无

毛blood旺乱改版-附鸡片鱼片焯水方法

Posted on 2021-10-25 Edited on 2021-12-20 In 热菜

备料

主料：毛肚、鸭blood，各种时令蔬菜

配料：鸡胸肉，鱼片，肉片，huotui片等

调料：郫县豆瓣，火锅底料，干辣椒，芫荽

做法

  1. 毛肚切丝或切片；
  2. 蔬菜焯水煮熟放在碗底；
  3. 肉片类过水焯熟（见备注）铺在蔬菜上；
  4. 葱姜蒜花椒（干辣椒可选）爆香锅底；加入郫县豆瓣炒香，同时加入火锅底料炒化；
  5. 加入水（或者高汤），大火煮开，捞出料渣弃置；
  6. 盐、白胡椒粉调味；
  7. 下入鸭blood煮熟；
  8. 下入毛肚，不超过15秒后关火；
  9. 倒入碗中，以水未没过全部食材为佳；
  10. 铺上生蒜、花椒面、花椒、辣椒面，油温6成，浇油；
  11. （如果有）锅内剩余汁水倒入碗内；
  12. 撒上芫荽，食用。

备注

  • 鸡片做法：鸡肉切片，用盐、味精、（一点点小苏打可选）一点料酒腌制10分钟左右，下沸水中焯熟即成。不要用酱油（如果是炒鸡或者炖鸡可以放酱油），不要勾芡；肉片同鸡片做法。
  • 鱼片做法：鱼肉请别人切成薄片，用盐、胡椒粉、一点料酒、葱、姜腌制10分钟左右，锅内烧水似开非开，鱼肉拍淀粉后逐一下入，煮至筷子夹着肉似断非断，捞出，口感最佳。
  • 严格说，正宗的毛blood旺，最后的辣椒应当是刀口辣椒而不是辣椒面。

附上10月24日作品，还不错。毕竟是当年学厨师的时候学的第一个大菜。

tips

  • 不用担心油多或者花椒面多，这道菜就要大麻大辣大油。
  • 肉片都是提前滑熟，做菜时不用放料酒；

粉蒸肉

Posted on 2021-10-18 Edited on 2021-10-25 In 热菜

备料

主料：鸡肉片（或其他猪、牛、羊均可）

配料：土豆、南瓜、红薯均可

调料：蒸肉米粉，葱花

做法

  1. 肉类洗净切片，腌制；
  2. 配料切滚刀块，沥干水分；
  3. 蒸肉米粉与切好的肉片拌匀；
  4. 盘子控干水分，盘底铺上切好的块茎类食物，拌好的蒸肉在上面均匀铺一层。
  5. 上锅蒸60分钟（高压锅时间减半）；
  6. 取出撒上葱花，食用

备注

蒸肉米粉可根据自己口味，适量添加红腐乳、辣椒面、红油、花椒面、胡椒面等调味料

tips

肉类只能放一层，多了蒸好后不易分开

炖鱼汤

Posted on 2021-10-08 In 热菜

备料

现杀河鱼（各种鱼类均可）

做法

鱼肉洗去血水，扣去腥鳃、牙齿，抽去腥线，刮鳞洗净粘液。

处理好的鱼肉加入姜、蒜、料酒、盐、白胡椒粉腌制

食用油润锅，撒入盐，升高温度后，下入鱼，鱼皮朝下煎熟；煎熟后加入葱姜蒜（可选花椒）炒香，加入水，炖鱼汤

想要奶白，就大火炖；想要清汤，就小火炖；火候越长越好。

用盐和胡椒粉调味。出锅前根据个人喜好，可选加入葱花、芫荽或香油。

备注

可加入豆腐、白菜、粉丝等材料，制成炖菜，同样鲜美可口。

tips

河鱼肉必须要新鲜的，才能鲜嫩不腥；如果鱼肉是现杀冻存的，只能保证不腥，但冻过后肉质变紧发硬，口感变差。
